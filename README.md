# FinOps

급여명세서 및 경비정산 자동화 프로젝트입니다. Python 기반 백엔드와 React 프론트엔드, Docker 인프라를 활용하여 사내 급여/경비 처리 업무를 자동화하는 것을 목표로 합니다.

## 프로젝트 개요

- **급여명세서 자동화**: 직원별 급여 데이터를 수집·계산하여 명세서를 자동으로 생성합니다.
- **경비정산 자동화**: 제출된 경비 내역을 검증하고 정산 처리 흐름을 자동화합니다.

## 기술 스택

### Frontend
- React

### Backend
- Python
- FastAPI
- pandas
- SQL

### Infra
- Docker

## 프로젝트 구조

```
FinOps/
├── frontend/   # React 기반 웹 클라이언트
├── backend/    # FastAPI 기반 API 서버 및 데이터 처리 로직
└── infra/      # Docker 관련 설정 파일
```
